Unidade de medida Absoluta

Px

Este texto tem um tamanho de 30px

O que faz: A medida não muda independente do elemento pai.

Unidade de medida Relativa

Por padrão, o body define a fonte de texto com 16px. Então, toda definição relativa desta página terá como base estes 16px.

Em

Este texto tem um tamanho de 2em

O que faz: Se baseia na medida do elemento pai e faz uma multiplicação simples. Por exemplo, um conteiner que possui como tamanho da fonte 16px, qualquer elemento dentro dele que tenha 1em possuirá o de tamanho 16px, ou se forem 2em então terá um tamanho de 32px e assim sucessivamente.

Rem

Este texto tem um tamanho de 3rem

O que faz: Mesmo comportamento do Em, mas ao invés de se basear no elemento pai, ele se baseia no elemento root (tag html).

Vw

Este texto tem um tamanho de 3vw

O que faz: Porcentagem da LARGURA relativa ao Viewport (espaço visível da página na tela).

  • Dica: Aumente/Diminua a largura da janela de seu navegador

Vh

Este texto tem um tamanho de 3vh

O que faz: Porcentagem da ALTURA relativa ao Viewport (espaço visível da página na tela).

  • Dica: Aumente/Diminua a altura da janela de seu navegador

%

Este texto tem um tamanho de 200%

O que faz: Porcentagem relativo a fonte definida no elemento pai.

Vmin

Este texto tem um tamanho de 3vmin

O que faz: Relativa a MENOR parte do Viewport, ou seja se a janela estiver com a largura menor que a altura, então será feita uma multiplicação simples tomando como base a largura do viewport naquele momento e vice versa.

Vmax

Este texto tem um tamanho de 2vmax

O que faz: Relativa a MAIOR parte do Viewport, ou seja se a janela estiver com a largura menor que a altura, então será feita uma multiplicação simples tomando como base a largura do viewport naquele momento e vice versa.
